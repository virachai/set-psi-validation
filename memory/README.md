# Memory Standards

All memory files in `memory/` must follow this standard format to ensure consistency and readability.

## Filename Convention

`YYYYMMDD-HHMMSS-slug-name.md`

## File Template

```markdown
# [Full Title]

## Metadata

- **Date:** YYYY-MM-DD
- **Author:** [Agent Name]
- **Status:** [DRAFT | COMPLETED]

## Context

A brief description of why this memory record exists.

## Changes Made

- Point 1
- Point 2

## Impact

- Point 1
- Point 2

## Next Steps

- Point 1
```
